工具名称：logo.img生成器（小米平板4专用）
作者：CK丶Tex

本工具用于自定义小米平板4的logo.img文件。
仅在小米平板4（MIUI 10.2 xiaomi.eu 8.11.1）上测试通过，其他机型尚未进行测试。

使用方法：把图片文件放在image目录下，运行程序。最后用fastboot将logo目录下生成的logo.img刷入即可。

注意事项：
1. 注意5个文件的文件名和后缀名，注意图片格式；
2. 01.png不能超过1535字节，02.bmp不能超过1050623字节，03.bmp不能超过187903字节，04.bmp不能超过2075135字节，05.bmp不能超过401407字节；
3. 强烈建议备份好原生logo.img，以防止替换失败或是不兼容。
